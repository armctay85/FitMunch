## TECHNICAL REQUIREMENTS
- **Device:** iPhone 15 Pro Max (6.7 inch display)
- **Resolution:** 1290 x 2796 pixels
- **Format:** PNG or JPEG, RGB color space
- **Number Required:** 5 minimum (can include more up to 10)
- **Localization:** English (Australia) primary
- **Mockup Style:** Use clean device frames with subtle shadow
- **Text Overlay:** Headline and sub-headline should be large, clear, and placed at top of screenshot
- **Background:** Use light gradient background that complements app colors

## CONTENT GUIDELINES
- Show real, appetizing food images (not generic icons)
- Use realistic data that looks authentic
- Ensure text is readable at App Store display sizes
- Highlight the app's core value in first 3 screenshots
- Show premium features in context (not just listed)
- Maintain consistent visual style across all screenshots
- Include both portrait and potentially landscape if relevant (though 6.7" is portrait)
