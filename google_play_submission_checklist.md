
# FitMunch Google Play Submission Checklist

## App Bundle/APK Preparation
- [ ] Build release AAB (Android App Bundle) using Android Studio
- [ ] Sign the app with a keystore (keep this safe for future updates)
- [ ] Test the signed APK on a real device

## App Store Listing Requirements
- [x] App icon (512x512px)
- [x] Feature graphic (1024x500px)
- [x] App screenshots (at least 2)
- [x] App description
- [x] Privacy policy URL: https://fitmunch.com.au/privacy
- [x] Content rating questionnaire completed

## Technical Requirements
- [ ] Target Android API level 33+ (Android 13+)
- [ ] Minimum Android API level 24 (Android 7.0+)
- [ ] App permissions justified in store listing
- [ ] Ensure subscription billing is properly tested
- [ ] Receipt validation working with Google Play
- [ ] Google Play signing enabled in Play Console

## Quality Checks
- [ ] App works offline or shows meaningful offline message
- [ ] UI displays correctly on different screen sizes
- [ ] No crashes or ANRs (Application Not Responding)
- [ ] Accessibility features implemented
- [ ] Consistent back-button behavior

## Final Steps
- [ ] Create a Google Play developer account ($25 one-time fee)
- [ ] Set up merchant account for in-app purchases
- [ ] Set up subscription products in Play Console
- [ ] Submit build to internal testing track first
- [ ] Complete all store listing information
- [ ] Submit for review
